英语语法

2023-05-11- 2023-05-12
学习-English

简单句

一个简单句只能有一个谓语动词，其他动词则为非谓语动词

基本句型

一般句子都可以看成主语+谓语

谓语动词是谓语的一部分

  1. 主语 + 不及物动词

  2. 主语 + 单及物动词 + 宾语

  3. 主语 + 双及物动词 + 间接宾语 + 直接宾语

  4. 主语 + 复杂及物动词 + 宾语 + 宾语补语

  5. 主语 + 系动词 + 主语补语/表语

    系动词用来赋予主语状态

句子成分

除了基本句型内的成分还有以下成分

  1. 定语

    主要用来修饰主语和宾语

    The little white是主语rabbit的定语

    a large是宾语carrot的定语

  2. 状语

    主要用来修饰谓语动词

    quickly就是ate的状语

  3. 同位语

    用来把主语或宾语重新说一遍

    an English与Rabbit同等地位，所以是同位语

  4. 谓语动词

    位于动词的词类只能是动词，其他成分的词类什么都有可能

    谓语动词的三大本领：

    （1）动作时间

    ​ 现在、过去、将来、过去将来

    （2）动作状态

    ​ 一般、完成、进行、完成进行

    （3）动作假设，情感…(语气)

    ​ 虚拟语气

    ​ If i were a rabbit…

    ​ 陈述语气

    ​ I ate a carrot and…

    ​ 祈使语气

    ​ Eat this carrot and…

    其中(1)(2)结合在一起就是时态。

  5. 非谓语动词

    非谓语动词几乎可以取代所有的从句

词类

相同的句子成分但词类不一定相同

  1. 名词

    人或物

    ribbit就是名称

  2. 冠词

    a为冠词

  3. 代词

    替代人和物

    I为代词

  4. 形容词

  5. 数词

  6. 副词

    修饰动词或形容词

    quickly为副词

  7. 介词

    表示和其他词关系的词

    with意思为”用“，是介词

  8. 叹词

    Ah是叹词

  9. 连词

    连接词和句

    because是连词

  10. 动词

  11. 助动词

    用助动词have 表示吃过了

    用助动词be 表示正在吃

    用助动词be 表示被吃

    用助动词be 表示有能力吃

    还有migt、must、do等等

复杂句

复杂句构成：主句 + 从句

包含关系：{主句{从句}}

从句

从句就是将简单句修改充当另外一个句子的成分。

将修改后的句子

作主语：主语从句

作宾语：宾语从句

作补语/表语：表语从句

作同位语：同位语从句

以上都是名词性从句

作定语：定于从句

做状语：状语从句

动词时态

将来时态

以下几种方式可以表达将来的意思

  • 一般现在时态表将来

  • 助动词be的变位 + going + to + 动词原形

    将be的变位改成was、were就是过去将来时态

  • 助动词be的变位 + to + 动词原形

    省略了going

  • 助动词be的变位 + about+ to + 动词原形

    about取代going

一般时态

一般只说明动作发生了，动作持续时间、何时开始、何时结束都是未知。

  • 一般现在时态

    直接为动词原形

    当为第三人称单数时，为动词原形+s

  • 一般过去时态

    直接+动词过去式

    过去式

    通常：+ed，如looked、smelled

    也要特殊的，如：

    eat、do、did -> ate、did、took

    bring、blow、begin、speak->brought、blew、began、spoke

  • 一般将来时态

    能够确定将来发生

    助动词will + 动词原形

  • 一般过去将来时态

    助动词would + 动词原形

进行时态

  • 过去进行时态

    助动词be的变位 + 动词的现在分词

    be的变位为was、were

    现在分词

    通常：+ing

    如looking、write->writing、die->dying

  • 现在进行时态

    助动词be的变位 + 动词的现在分词

    be的变位为am、are、is

  • 将来进行时态

    will + be + 动词的现在分词

  • 过去将来进行时态

    would + be + 动词的现在分词

完成时态

  • 现在完成时态

    助动词have的变位 + 动词的过去分词

    过去分词

    通常：+ed

    如looked、smelled、eat->eaten、bring->brought、feel->felt

    eat、give、see、fall都是+en

    bring、buy、fight、think->brought、bought、fought、thought

    feel、keep、sleep、leave->felt、kept、slept、left

  • 过去完成时

    助动词have的变位 + 动词的过去分词

  • 将来完成时

    will + have + 动词的过去分词

  • 过去将来完成时

    would + have + 动词的过去分词

完成进行时态

  • 现在完成进行时

    have的变位 + been +动词的现在分词

  • 过去完成进行时

    have的变位 + been +动词的现在分词

  • 将来完成进行时

    will + have + been +动词的现在分词

  • 过去将来完成进行时

    would + have + been +动词的现在分词

    我说过，到明天下午一点，我已经吃了两个小时的午饭了。

would表虚拟语气
