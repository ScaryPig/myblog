命令执行函数

PQexec

提交一个命令给服务器并且等待结果

PQexecParame

PQexecParams与PQexec相似，但是提供了额外的功能：参数值可以与命令字符串分开指定，并且可以以文本或二进制格式请求查询结果。3.0协议后才支持

PQprepare

允许将被重复使用的命令只被解析和规划一次，而不是在每次被执行时都被解析和规划。

PQresultStatus

返回该命令枚举类型的结果状态。

PQresStatus可以将返回的枚举转换成描述状态编码的字符串常量。

可以PQerrorMessage，得到错误的详细信息

PQclear

释放与一个PGresult相关的存储，每一个命令结果不再需要时应该用PQclear释放。

查询结果信息

内容信息

PQgetvalue

返回一个PGresult的一行的单一域值。行和列号从 0 开始。空值返回一个空字符串

PQgetisnull

测试一个域是否为空值。行号和列号从 0 开始。

如果该域是空，这个函数返回 1，否则为0

PQgetlength

返回一个域值的真实长度，以字节计。行号和列号从 0 开始。

PQntuples

返回查询结果中的行（元组）数

PQnfields

返回查询结果中每一行的列（域）数

PQfname

返回与给定列号相关联的列名。列号从 0 开始。如果列号超出范围，将返回NULL。

PQfnumber

返回与给定列名相关联的列号。如果给定的名字不匹配任何列，将返回 -1。

单行模式
