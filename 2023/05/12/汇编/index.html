汇编语言

2023-05-12- 2023-05-25
学习-汇编

1字 == 2字节 == 2Byte == 16位 == 16bit

寄存器

一个16位寄存器大小为16bit，也就是2Byte，但最大能存2^16也就是64KB。

8086CPU有14个寄存器：通用寄存器（AX累加寄存器、BX基址寄存器、CX计算寄存器、DX数据寄存器）、SI源变址、DI目的变址、SP栈顶偏移地址、BP基址指针、IP偏移地址、段寄存器（CS代码段、SS堆栈段、DS数据段、ES附加段）、PSW。

段寄存器不能直接放入，要通过通用寄存器

为了兼容8位寄存器，AX又可以分为两个独立的8位寄存器（AH、AL），其他通用寄存器也同样可分。

mul

两个8位数相乘，一个指出，另一个默认在AL中，结果放入到AX中

两个16位数相乘，结果高位存放在DX，低位存放在AX

div

AX存放16位被除数，BL存放8位除数，结果存放在AL中，余数存放在AH中

AX ÷ BL = AL ······ AH

DX存高位，AX存低位被除数，BX存放16位除数，余数存放在DX中

DX AX ÷ BX = AX ······ DX

BX也可以替换成其他寄存器

xor

mov三个字节，xor两个字节

and、or

and：二进制位都为1时结果为1，否则为0

or：二进制位都为0时结果为0，否则为1

结果会放到前面的寄存器al内

栈

高地址栈底，低地址栈顶

SS:SP指向栈顶

  • 10000~1000F为栈空间，当栈空时，SS:SP指向10010
  • 一个栈最大为2^1=64K。

push

先sp = sp - 2，再入栈

pop

先出栈，再sp = sp + 2

shl、shr

shr右移

rol、ror

rol循环左移，将最左边二进制位放到最右边

ror循环右移，将最右边二进制位放到最左边

Debug

  • R查看、改变CPU寄存器的内容

  • D查看内存中的内容

  • E改写内存中的内容

  • U将内存中的机器指令翻译成汇编指令

  • T执行一条机器指令

    在Debug中只认16进制：00F1，不能加H。

  • A以汇编指令的格式在内存中写入一条机器指令

  • Q退出Debug

  • G代码执行到CS:0012停止

  • P完成loop执行
